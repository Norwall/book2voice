# Локальная озвучка книг

Streamlit-инструмент для генерации русских аудиокниг через Silero TTS на CPU.

## Установка

```bat
install.bat
```

Первый запуск скачает Python-зависимости и модель Silero. После этого генерация работает локально из кэша.

## Запуск интерфейса

```bat
run_streamlit.bat
```

Откройте адрес, который покажет Streamlit, загрузите `EPUB`, `TXT` или `FB2`, выберите голос и нажмите `Сгенерировать`.

Главы сохраняются как `001.mp3`, `002.mp3`, `003.mp3` и так далее. Если включить `Склеить в одну книгу`, рядом появится `book.mp3`.

Во время генерации доступна кнопка `Стоп`. Остановка выполняется безопасно: текущий фрагмент озвучки или операция конвертации завершается, после чего генерация прекращается. В параметрах можно выбрать, что делать после остановки: оставить уже готовые главы или удалить папку результата.

## Запуск из командной строки

```bat
python generate_book.py book.epub --voice baya --out outputs\my_book
python generate_book.py book.txt --voice xenia --merge
python generate_book.py book.fb2 --voice baya --out outputs\fb2_book
```

Доступные голоса: `aidar`, `baya`, `kseniya`, `xenia`, `eugene`.

## Примечания

- Для MP3 используется FFmpeg из пакета `imageio-ffmpeg`; отдельная установка FFmpeg не нужна.
- Если TXT не содержит заголовков вида `Глава ...`, файл режется на блоки заданного размера.
- Если FB2 не содержит секций, текст `body` режется на блоки заданного размера.
- Кнопка `Стоп` работает в интерфейсе Streamlit. Запуск из командной строки завершается обычным прерыванием процесса.
- Для длинных книг CPU-генерация может занимать часы.
